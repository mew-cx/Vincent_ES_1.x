src Folder
----------

This folder contains the source files for building the graphics library. The
source files are organized into the following structure:

src/
	config.h		Library configuration settings
	egl/			Implementation of egl API
	gl/			Implementation of gl API
	platform/		Platform abstraction layer
	processor/		Virtual processor for shader execution
	raster/			Rasterization of geometric primitives
	render/			Geometry processing functions
	slang/			Shader language implementation
